외롭다는 것은 새벽녘 희뿌연 가로등과 같은 것정년퇴직후 아무도 찾아 오지 않는 것이국만리 낯선 땅에서 혼자 밥 먹는 것윗목 한 귀퉁이에 있는 보릿자루 같은 것곰팡이 냄새가 스멀거리는 신발장 같은 것늦가을 대롱대롱 매달려 있는 까치밥 같은 것늦은 귀가시간에 반겨주는 사람 하나 없는 것자취방에서 김치없이 홀로 라면을 끓여 먹는 것선술집에서 홀로 한 잔의 막걸리를 걸치고 있는 것함께 투쟁하자 해놓고 결정적일 나몰라라 돌아서는 것식사 한 번,사랑 한 번 청을 못받은 홀어미 신세 같은 것초상집에 신발 몇 개 있고 조화 한 개 조등을 밝히고 있는 것 이런 것을 시라고 쓰고 있는, 어쩌면 너무나도 매너리즘 같은 것
